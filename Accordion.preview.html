Accordion

Props

Prop Type Default Description
title string "How can I make a booking?" Header text shown in the collapsed and expanded state
body string "Just select a service…" Body text revealed when the accordion is open
icon ReactNode undefined Optional leading icon displayed before the title
cta { label, onClick? } undefined Optional call-to-action button shown at the bottom of the expanded body
defaultOpen boolean false Controls whether the accordion starts in the open state
